📖 Proje Hakkında

Bu proje, modern yazılım geliştirme prensiplerine uygun olarak geliştirilmiş bir .NET 8 web uygulamasıdır.
Temel amacı; modüler yapı, sürdürülebilir mimari ve NoSQL veri yönetimi konularında pratik bir örnek sunmaktır.

🚀 Kullanılan Teknolojiler
Teknoloji	Açıklama
🟣 .NET 8	Modern backend geliştirme
🌐 ASP.NET Core MVC	Web uygulama framework’ü
🍃 MongoDB	NoSQL veri tabanı
🔁 AutoMapper	DTO dönüşümleri
🔌 Dependency Injection	Gevşek bağlı mimari
🧱 Generic Repository	Veri erişim soyutlama
🧩 ViewComponent	Modüler UI yapısı
🗂️ Area	Admin ve kullanıcı ayrımı
🧠 Mimari Yapı

🗂️ Area yapısı → Admin ve kullanıcı tarafının ayrıştırılması

🧩 ViewComponent → Yeniden kullanılabilir UI bileşenleri

🧱 Generic Repository → Temiz ve sürdürülebilir veri erişimi

🔌 Dependency Injection → Gevşek bağlı yapı (loose coupling)

🔁 AutoMapper → DTO ↔ Entity dönüşümleri

⚙️ Özellikler

🛠️ Admin panel üzerinden veri yönetimi

🍃 MongoDB ile esnek veri saklama

🧩 Modüler ve yeniden kullanılabilir UI bileşenleri

📦 Temiz ve okunabilir kod organizasyonu
